# 部署物生成命令

部署物可以通过部署命令在您的位置生成，由此生成的部署物不属于任何阵营。

当前版本：[V4.5](/update/v4.5/release-notes)

!!! warning "Squad 仅允许在**自定义服务器**或**本地服务器**中生成部署物。"

!!! warning "生成的部署物不服从重力。"
