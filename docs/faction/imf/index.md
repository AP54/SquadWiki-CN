# [IMF]非正规民兵部队 | Irregular Militia Forces

## 简介

这支由自由战士、准军事游击队和逃兵组成的非常规部队在东欧平原上游荡，所到之处都造成混乱。他们主要由武器和毒品运输提供资金，拥有大量武器和设施，使他们成为一支强大的战斗力量。
