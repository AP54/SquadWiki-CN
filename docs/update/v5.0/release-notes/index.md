## 系统与玩法更新

    新增了隐私条款提示

    调整了俄罗斯陆军沙漠迷彩、美国陆军沙漠迷彩和美国陆军林地迷彩的制服样式，使其与其他派别的外观更加一致。

    更新了训练界面的菜单，包括两个额外的选项，允许玩家在太平洋训练场和詹森靶场测试人民解放军海军陆战队和俄罗斯空降军派系。增加了滚动条，以适应所有詹森靶场的变化，这是在这个菜单重新设计之前的一个临时解决方案。

    在用户界面中添加了代表机动火炮系统载具的图标。

    更新了M16A2及其变种的描述。

## 一般BUG修复

    修正了一个可能导致切换地图后立即听到巨大点击声的错误

    修正了一个与武器音效导致音频系统过载相关的客户端崩溃问题

    修正了一个可导致指挥官徐进炮击区在小地图上的攻击宽度指示器看起来比实际的攻击更窄的问题。

    修正了在小地图上的其他标记周围放置引导线时使得标记无法被删除的一个用户界面问题。

    修正了服务器浏览器中的一个用户界面问题，当你运行与服务器相同的游戏版本时，在 "模式 "一栏中将不再包含游戏版本号。

    修正了一个允许将可部署物放置在车辆之上的错误，即使车辆已经离开，这些可部署物也会留在原地。

    修正了一个UI错误，当加入一个包含玩家未安装模组的模组服务器时，该错误可能会阻止服务器浏览器中的模组详情界面弹出。

    修正了中国人民解放军ZTZ99A主战坦克右侧履带的视觉问题。

    修正了士兵的武器在穿过某些迷彩网时会被强制降低的问题。

    修正了一个UI错误，该错误可能导致在与弹药箱互动后，角色请求按钮与径向菜单上的车辆请求按钮重叠。

    修正了非正规民兵 M16A2步枪的机械瞄具对齐问题，现在它们应该对齐了。

    修正了解放军ZBD04A步兵战车的炮塔指示器在车长位上出现错误的一个小的UI问题。

    修正了一个小的视觉问题，该问题导致美国M1126 M2装甲运兵车后面的天线加载不正确，看起来更像一棵植物而不是天线。

    修正了一个允许非正规民兵的步兵无限制地放置沙袋的错误，而不是预期的每个玩家10个的限制。
