# TA | 装甲死斗

> 文章作者：桀氓AlbertWensley

!!! failed "装甲死斗模式仅适于人数极少时暖服和训练使用，不适用于人数较多的对局"

装甲死斗是 Squad 中一种用于训练的模式，双方阵营完全镜像，配有武装载具。

双方的游戏目标是率先全部击毁敌方载具。
