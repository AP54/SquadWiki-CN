# [MEA]中东联盟 | Middle Eastern Alliance

## 简介

中东联盟是一个军事组织，旨在协调该地区一些结盟国家之间的资源和国防政策。他们使用训练有素的本土人员和外国武器储备，这意味着他们非常适合与技术最先进的对手作战。
