这里列出了所有在战术小队中可玩的派系，游戏里大多数的拍戏是基于现实世界的武装部队；少数几个是真实世界中合并的或具有代表性的军事团体。

各派系有属于自己的武器和车辆。一些派系还部署在固定类型的的地图。例如[中东联盟](./MEA/)和[反叛军](./INS/)最常见在中东和南亚地图。
