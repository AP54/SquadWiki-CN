# [USA]美国陆军 | US Army

## 简介

美国陆军是美国武装部队的陆战部队。它是世界上最大的军事组织之一，负责美军的陆上行动。美国陆军由于广泛参与战争而积累了丰富的经验，也是世界上技术最先进的军队之一。

《战术小队》中代表美国陆军的旗帜是星条旗。它由十三条红白相间的水平条纹组成（代表最初的十三个殖民地），一个蓝色矩形中有 50 颗星（代表美国各州）。
