# [RGF]俄罗斯陆军 | Russian Ground Forces

## 简介

俄罗斯军队的陆地部分。今天的俄罗斯军队是在苏联解体后由旧苏联军队组成的，是一支高度专业化、技术现代化的战斗部队，能够迅速部署到海外。
